# 勇者通關手冊
這是一個基於C++開發的文字RPG遊戲，玩家可以選擇不同的職業，透過打怪、賺取金幣、在商店購買道具來強化自己，並在隨機事件中考驗運氣，最終目標是擊敗山頂的終極Boss**哥吉拉**！

## 遊戲特色
* **三大職業**：戰士（血厚）、射手（高普攻）、法師（高爆發），每個職業皆有專屬雙技能。
* **回合制戰鬥**：考驗玩家對血量與魔力的控管。
* **豐富的裝備搭配**：透過購買「能量飲尿」、「麻辣加血」或「超級馬力」，打造專屬於自己的攻擊流派。
* **命運隨機事件**：過場途中可能遭遇小偷、腳滑跌倒、淋雨感冒，或是幸運撿到錢包觸發隱藏加碼商店！

## 遊戲介紹
### **角色職業及技能**
* 戰士(血量：100 / 普通攻擊：20 / 魔量：120)，技能一(劈砍  魔量：-30 / 攻擊：30)、技能二(正義之劍  魔量：-50 / 攻擊：50)
* 射手(血量：75 / 普通攻擊：30 / 魔量：100)，技能一(一箭穿心  魔量：-25 / 攻擊：40)、技能二(萬箭齊發  魔量：-40 / 攻擊：60)
* 法師(血量：50 / 普通攻擊：40 / 魔量：80)，技能一(火球  魔量：-20 / 攻擊：60)、技能二(迷幻法陣  魔量：-35 / 攻擊：70)

### **裝備介紹**
* 能量飲尿(價格：10，功能：魔量+20/每輪)
* 麻辣加血(價格：20，功能：最大血量+20/每個、血量+15/每輪)
* 超級馬力(價格：30，功能：普通攻擊+20/每輪)

### **怪物陣容**
* 哥布林(血量：60 / 傷害：20 / 擊殺金幣：+10)
* 哥布林長老(血量：70 / 傷害：15 / 擊殺金幣：+20)
* 石頭人(血量：80 / 傷害：20 / 擊殺金幣：+10)
* 閃電鳥(血量：100 / 傷害：30 / 擊殺金幣：+20)
* 哥吉拉(血量：150 / 傷害：50 / 擊殺獲勝!!!)

## 遊玩方式
* 只需透過鍵盤輸入對應的數字操作角色。
* 使用技能時會消耗對應魔量，當魔量不足時僅能使用普通攻擊。
* 擊殺怪物可以獲得對應金幣，金幣可以在擊殺怪物後於商店使用，金幣可保存在包包中。
* 於商店購買裝備後，將在下次對戰前直接生效。
* 擊敗哥吉拉獲勝，請小心應付。
* 路程當中有機率遇到隨機事件，加油吧勇者們!!!祝您一路順風~~

## 如何編譯與執行

本專案將類別拆分為多個 `.h` 與 `.cpp` 檔案，請確保所有檔案皆在同一個資料夾目錄下。
